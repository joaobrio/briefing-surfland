            <h2 class="text-2xl font-semibold text-sky-700 mb-4">1. Snapshot Executivo</h2>

            <p class="mb-4 text-gray-700">Esta seção apresenta uma visão geral da Surfland Brasil, incluindo sua localização, segmento de atuação e um resumo do seu posicionamento no mercado. É o ponto de partida para entender a essência do negócio e sua proposta de valor única.</p>

            

            <div class="mb-6">

                <h3 class="text-xl font-semibold text-slate-700 mb-2">Empresa:</h3>

                <p class="text-gray-600">Surfland Brasil, localizada em Garopaba, Santa Catarina.</p>

            </div>

            <div class="mb-6">

                <h3 class="text-xl font-semibold text-slate-700 mb-2">Segmento:</h3>

                <p class="text-gray-600">Resort de surf com modelo de multipropriedade, direcionado ao turismo de experiência de alto padrão e entretenimento familiar. Este segmento combina investimento imobiliário (cotas de multipropriedade) com hotelaria de lazer e uma oferta de entretenimento única centrada na prática do surf.</p>

            </div>

            <div>

                <h3 class="text-xl font-semibold text-slate-700 mb-2">Contexto Rápido:</h3>

                <div id="contextoRapidoContent" class="space-y-3 text-gray-600">

                    <p>A Surfland Brasil se estabelece no mercado com um posicionamento distintivo e ambicioso, autointitulando-se o "primeiro clube e resort em multipropriedade no Brasil" e, de forma ainda mais audaciosa, "o primeiro resort de surf com multipropriedade do mundo". O cerne de sua atratividade reside na sua piscina de ondas com tecnologia Wavegarden, uma inovação que promete "surf sem limites, independentemente da estação", democratizando o acesso a ondas perfeitas durante todo o ano.</p>

                    <p>O modelo de negócio centraliza-se na comercialização de cotas de multipropriedade, que conferem aos seus detentores benefícios como acesso exclusivo às instalações do clube e resort, flexibilidade na utilização ao longo do ano e a constituição de um patrimônio vitalício e com potencial de valorização. Para além da experiência de surf, o complexo da Surfland Brasil foi concebido para oferecer um leque abrangente de opções de lazer e entretenimento para toda a família. Estas incluem múltiplas piscinas, diversas quadras esportivas, um skate park, um museu do surf, um hotel com 78 leitos, spa, seis variadas opções gastronômicas e áreas de recreação infantil dedicadas.</p>

                    <p>O posicionamento da marca é cuidadosamente construído sobre pilares de exclusividade, sofisticação e inovação, com um forte apelo à vivência autêntica do surf e ao fortalecimento dos laços familiares. Adicionalmente, a Surfland Brasil demonstra um compromisso com a sustentabilidade em seu projeto, incorporando elementos como telhados verdes, o uso de edifícios pré-fabricados para minimizar o impacto da construção, a instalação de painéis solares para geração de energia limpa, sistemas de captação e reuso de água da chuva e a integração da flora nativa na paisagem do empreendimento.</p>

                    <p>A natureza multifacetada da oferta da Surfland – que engloba desde o investimento em multipropriedade até a estadia em um resort de luxo e o acesso a uma ampla gama de atividades de lazer – sugere um volume considerável e diversificado de interações com potenciais clientes. Cada uma dessas ofertas atrai perfis de clientes distintos, desde o investidor focado no retorno e na valorização patrimonial, passando pelo turista de luxo que busca experiências exclusivas, até famílias que priorizam o entretenimento e surfistas dedicados à prática do esporte. Esta diversidade de públicos e produtos naturalmente resulta em um fluxo elevado de questionamentos e geração de leads através de múltiplos canais de comunicação. Consequentemente, a gestão eficiente, ágil e personalizada desta demanda representa um desafio operacional de grande magnitude para a empresa.</p>

                    <p>Ademais, o forte componente de "experiência" e "exclusividade" que permeia o posicionamento da Surfland eleva consideravelmente as expectativas dos clientes por um atendimento que seja igualmente premium e personalizado, desde o primeiro contato. Clientes que buscam e estão dispostos a pagar por tais vivências diferenciadas tendem a ser menos tolerantes a respostas lentas, genéricas ou impessoais. O contato inicial, frequentemente estabelecido por meios digitais como o WhatsApp, funciona como uma extensão da promessa da marca. Se este primeiro atendimento não refletir a "Essência" premium do empreendimento, conforme preconizado pelo Framework E³, a credibilidade da promessa de uma "experiência única" pode ser irremediavelmente abalada. Neste contexto, a qualidade do atendimento inicial torna-se um fator tão crítico para a conversão e satisfação do cliente quanto a própria qualidade da piscina de ondas.</p>

            // Gemini API Integration

            const apiKey = ""; // API Key will be injected by the environment


            async function callGeminiAPI(prompt, outputElement, loadingElement) {

                loadingElement.style.display = 'inline';

                outputElement.style.display = 'none';

                outputElement.textContent = '';


                const chatHistory = [{ role: "user", parts: [{ text: prompt }] }];

                const payload = { contents: chatHistory };

                const apiUrl = `https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=${apiKey}`;


                try {

                    const response = await fetch(apiUrl, {

                        method: 'POST',

                        headers: { 'Content-Type': 'application/json' },

                        body: JSON.stringify(payload)

                    });


                    if (!response.ok) {

                        const errorData = await response.json();

                        throw new Error(`API Error: ${response.status} ${response.statusText}. Details: ${JSON.stringify(errorData)}`);

                    }


                    const result = await response.json();


                    if (result.candidates && result.candidates.length > 0 &&

                        result.candidates[0].content && result.candidates[0].content.parts &&

                        result.candidates[0].content.parts.length > 0) {

                        const text = result.candidates[0].content.parts[0].text;

                        outputElement.textContent = text;

                        outputElement.style.display = 'block';

                    } else {

                        outputElement.textContent = 'Não foi possível obter uma resposta da IA ou a resposta está vazia.';

                        outputElement.style.display = 'block';

                    }

                } catch (error) {

                    console.error('Erro ao chamar a API Gemini:', error);

                    outputElement.textContent = `Erro ao conectar com a IA: ${error.message}. Verifique o console para mais detalhes.`;

                    outputElement.style.display = 'block';

                } finally {

                    loadingElement.style.display = 'none';

                }

            }
